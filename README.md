# 분석

## 1. 환율 × 입국 목적 민감도 분석

### 설명
각 목적이 환율에 얼마나 민감한지 상관관계를 측정

### 가설
- 질병치료는 "필요 방문"이므로 환율 비탄력적일 것
- 관광은 환율 탄력적(원화 약세 → 외국인에게 저렴 → 관광 증가)

---

## 2. 입국 목적 구성비 × 1인당 관광수입의 관계

### 가설
- 사업·질병치료 같은 고단가 방문객 비율이 높은 달에 1인당 관광수입도 높아질 것

---

## 3. 환율 × 1인당 관광수입

### 가설
- 원화 가치가 오를 수록 1인당 관광수입이 증가할 것

---

# 인사이트

## 1.환율 x 입국 목적 민감도 분석

### 가)
환율은 외래관광객수에 영향을 미친다

### 나)
상관계수와 탄력성으로 확인하였을 때 원화 약세시 관광 목적 방한외국인 증가, 질병치료는 0에 가깝게 영향 받지 않는다.

### 다)
입국 목적별 산점도는 회귀선의 기울기로 연동성을 알 수 있다. 관광의 경우 회귀선이 가파른 우상향이지만 질병치료는 보다 완만하다.

### 결론
가설대로 관광 목적의 방문은 환율의 영향을 많이 받지만 질병 치료등 필수 방문객은 환율에 거의 영향을 받지 않았다.

---

## 2.입국 목적 구성비 × 1인당 관광수입의 관계

### 가)
월별 입국 목적 중 관광이 대다수를 차지한다

### 나)
상관계수의 양에 있는 막대는 관광과 방문으로 사업과 질병치료 목적의 방문객이 많은 달이 고단가라 예측한 것과 달랐다.

### 다)
고단가 추정비율 그래프에서도 사업,질병,회의 방문객이 많은 달이 1인당 평균 지출이 낮은 것을 확인할 수 있었다.

### 결론
관광에서 사용하는 지출이 사업, 회의 같은 출장자의 지출보다 크다 예상하였지만 결과는 반대로 관광 목적의 방문객이 많을 때 1인당 관광수입이 컸다. 다만, 한국 방문 비수기 등 여러 인과 요인이 있을 것으로 예상된다.

---

## 3.환율 × 1인당 관광수입

### 가)
환율과 1인당 관광수입 시계열을 통해 확인했을 때, 비슷한 방향도 있지만 일부 구간에서는 어긋남을 보였다.

### 나)
산점도를 통해 같은 달의 환율과 관광수입 관계를 비교해보고, 시차 막대 그래프를 통해 환율이 시차를 두고 반영되는지 확인해 보았으나 유의미한 인사이트는 도출하기 어려웠다.

### 결론
시계열 그래프에서 확인했듯이, 환율이 오를 수록 1인당 관광 수입이 약하게 늘어나는 것 같지만, 그래프의 뚜렷한 방향성을 찾기 어려워 가설 검증은 불가했다.
